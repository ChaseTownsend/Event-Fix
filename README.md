# What does this plugin do?
#### TLDR : Fixes the Maximum damage shown (32,767 -> 2,147,483,647).
This Plugin Fixes the Visual Damage events being capped at 32,767 (16-bits), and increases that cap to 2,147,483,647 (32-bits).
<br><br>

> [!IMPORTANT]
> **Reload Map or Restart Server after adding plugin and data For Plugin to take Effect**
<br>

> [!WARNING]
> **There is currently No Windows 64 Signature for `CGameEventManager::CreateEvent`**
